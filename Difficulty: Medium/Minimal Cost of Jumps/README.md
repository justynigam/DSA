<h2><a href="https://www.geeksforgeeks.org/problems/minimal-cost/1">Minimal Cost of Jumps</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="color: #273239; font-family: Nunito, sans-serif;"><span style="font-size: 18px; letter-spacing: 0.162px; background-color: #ffffff;">Y</span></span><span style="font-size: 14pt;">ou are given an array <strong>arr[]</strong> of height of n stones and an integer <strong>k</strong>, where arr[i] represents the height of the i-th stone. You start at index 0 and want to reach the last stone (index n-1). </span><span style="font-size: 14pt;">From any index i, you can jump to any index j such that </span><span style="font-size: 14pt;">i &lt; j ≤ i+k. </span><span style="font-size: 14pt;">The cost of jumping from stone i to stone j is <strong>abs</strong></span><strong><span style="font-size: 14pt;">(arr[i] − arr[j]).</span></strong><span style="font-size: 14pt;"> Find the minimum total cost required to reach the last stone.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong> </span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> k = 3, arr[] = [10, 30, 40, 50, 20]<br></span><span style="font-size: 18px;"><strong>Output:</strong> 30<br></span><span style="font-size: 18px;"><strong>Explanation:</strong> </span><span style="font-size: 14pt;">An optimal path is from index 0-&gt;1-&gt;4. The minimum possible total cost is calculated as |10−30| + |30−20| = 20+10 = 30.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> k = 1, arr[] = [10, 20, 10]<br></span><span style="font-size: 18px;"><strong>Output:</strong> 20<br></span><span style="font-size: 18px;"><span style="font-size: 14pt;"><strong>Explanation:</strong> An optimal path is from index 0-&gt;1-&gt;2. The minimum possible total cost is calculated as |10−20| + |20−10| = 10+10 = 20.</span><br></span></pre>
